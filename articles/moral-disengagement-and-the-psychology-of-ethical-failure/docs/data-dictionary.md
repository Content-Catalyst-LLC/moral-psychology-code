# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `case_id` | Synthetic case identifier |
| `moral_justification` | Synthetic higher-purpose or necessity framing |
| `euphemistic_language` | Synthetic sanitized or distancing language |
| `advantageous_comparison` | Synthetic comparison with worse conduct or alternatives |
| `responsibility_shifting` | Synthetic displacement or diffusion of responsibility |
| `consequence_distortion` | Synthetic minimization, delay, abstraction, or distortion of consequences |
| `dehumanization` | Synthetic reduction of perceived target humanness or moral standing |
| `victim_blaming` | Synthetic attribution of responsibility to harmed persons or groups |
| `anticipated_guilt` | Synthetic anticipated guilt or internal self-sanction |
| `institutional_pressure` | Synthetic pressure from organization, group, role, target, or incentive system |
| `consequence_visibility` | Synthetic visibility of human impact |
| `victim_recognition` | Synthetic recognition of harmed person dignity, standing, and testimony |
| `accountability_strength` | Synthetic accountability, review, and answerability strength |
| `moral_disengagement` | Synthetic moral disengagement score |
| `action_latent` | Latent synthetic unethical action score |
| `unethical_action_probability` | Synthetic probability of unethical action |
| `unethical_action` | Binary synthetic unethical action indicator |
| `disengagement_group` | Quartile grouping of moral disengagement |
