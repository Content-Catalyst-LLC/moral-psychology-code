# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `case_id` | Synthetic case identifier |
| `perceived_harm` | Synthetic recognition of injury, threat, vulnerability, or damage |
| `affective_activation` | Synthetic emotional force that makes the case vivid |
| `social_salience` | Synthetic degree to which the case is socially prominent, identity-relevant, or publicly charged |
| `principled_reasoning` | Synthetic use of moral principles, consistency tests, and normative commitments |
| `evidential_reconsideration` | Synthetic openness to facts, counterevidence, and uncertainty |
| `contextual_knowledge` | Synthetic information about history, institutions, consequence, and situation |
| `cultural_framing` | Synthetic interpretive moral vocabulary inherited from social worlds |
| `group_identity_pressure` | Synthetic pressure from group belonging, identity, or loyalty |
| `time_for_reflection` | Synthetic time or space available for deliberation |
| `institutional_review` | Synthetic procedural review, evidence testing, or formal deliberation |
| `intuitive_appraisal` | Synthetic rapid first-pass moral evaluation |
| `reflective_evaluation` | Synthetic later structured moral evaluation |
| `intuition_weight` | Synthetic relative weight assigned to intuition in final judgment |
| `final_moral_evaluation` | Synthetic final moral evaluation after intuitive and reflective combination |
| `reflective_revision` | Synthetic difference between reflective evaluation and intuitive appraisal |
| `revision_magnitude` | Synthetic absolute size of reflective revision |
