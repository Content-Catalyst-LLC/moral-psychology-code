# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `person_id` | Synthetic person identifier |
| `org_id` | Synthetic organization identifier |
| `moral_identity` | Synthetic integration of moral commitments into self-concept |
| `empathic_concern` | Synthetic concern for suffering, vulnerability, and need |
| `self_regulation` | Synthetic capacity for restraint, consistency, and action under pressure |
| `social_desirability` | Synthetic tendency to present oneself favorably |
| `moral_salience` | Synthetic visibility and urgency of moral features |
| `moral_judgment` | Synthetic explicit moral endorsement or judgment |
| `moral_action` | Synthetic enacted moral conduct indicator or score |
| `judgment_action_gap` | Synthetic difference between moral judgment and moral action |
| `institutional_pressure` | Synthetic pressure that suppresses moral action |
| `accountability_climate` | Synthetic institutional support for accountability |
| `repair_infrastructure` | Synthetic availability of apology, restitution, redesign, and follow-up |
| `retaliation_risk` | Synthetic risk attached to truth-telling or moral action |
| `moral_visibility` | Synthetic contextual visibility of morally relevant features |
| `action_probability` | Synthetic probability of moral action in an agent-based simulation |

## Generated outputs

| Output | Description |
|---|---|
| `moral_psychology_simulated_individual_data.csv` | R-generated multilevel synthetic dataset |
| `moral_psychology_organizational_summary.csv` | R organization-level moral climate summary |
| `moral_psychology_gap_model_coefficients.csv` | R fixed-effect model coefficients |
| `moral_psychology_gap_model_fit.csv` | R model fit summary |
| `moral_psychology_gap_prediction_grid.csv` | R prediction grid |
| `moral_psychology_agent_simulation.csv` | Python agent-based moral action simulation |
| `moral_psychology_pressure_summary.csv` | Python summary by institutional pressure level |
| `moral_psychology_time_summary.csv` | Python time-series summary |
| `moral_psychology_risk_report.csv` | Python high-risk moral environment report |
| `moral_psychology_high_gap_cases.csv` | Python high judgment-action gap cases |
