# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `employee_id` | Synthetic employee identifier |
| `ethical_sensitivity` | Synthetic ethical attention and sensitivity indicator |
| `role_constraint` | Synthetic role constraint indicator |
| `cultural_climate` | Synthetic organizational ethical climate indicator |
| `performance_pressure` | Synthetic target, output, revenue, or speed pressure |
| `hierarchy_intensity` | Synthetic authority-gradient indicator |
| `diffusion_responsibility` | Synthetic diffusion of responsibility indicator |
| `accountability_strength` | Synthetic accountability clarity and enforceability indicator |
| `speak_up_protection` | Synthetic protection for dissent and ethical warning |
| `value_alignment` | Synthetic alignment between mission and incentives |
| `corruption_pressure` | Synthetic corruption, capture, or purpose-distortion pressure |
| `moral_decision_tendency` | Synthetic moral decision tendency score |
| `unethical_probability` | Synthetic probability of unethical action |
| `unethical_action` | Binary synthetic unethical-action indicator |
| `institutional_integrity` | Synthetic institutional integrity score |
| `accountability_band` | Quartile grouping of accountability strength |
