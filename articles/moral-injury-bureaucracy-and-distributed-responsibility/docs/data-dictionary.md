# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `worker_id` | Synthetic worker or role identifier |
| `conscience_conflict` | Synthetic conflict between moral standards and required action |
| `institutional_betrayal` | Synthetic betrayal by authority, leadership, peers, or institution |
| `distributed_responsibility` | Synthetic diffusion of responsibility across roles and procedures |
| `morally_injurious_exposure` | Synthetic exposure to morally injurious conditions |
| `scarcity_constraint` | Synthetic resource, staffing, or capacity constraint |
| `accountability_clarity` | Synthetic clarity of agency, authority, escalation, and responsibility |
| `organizational_opacity` | Synthetic opacity of rules, causation, or decision pathways |
| `protected_voice` | Synthetic protected ability to raise ethical concerns |
| `repair_capacity` | Synthetic institutional ability to acknowledge and repair harm |
| `moral_injury_risk` | Synthetic moral injury risk value |
| `elevated_moral_injury` | Binary synthetic indicator based on risk threshold |
| `accountability_band` | Quartile grouping of accountability clarity |
| `many_hands_pressure` | Synthetic index combining distributed responsibility and opacity |
