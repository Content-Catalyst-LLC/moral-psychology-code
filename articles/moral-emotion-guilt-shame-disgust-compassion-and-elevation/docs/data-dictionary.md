# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `case_id` | Synthetic case identifier |
| `perceived_wrongdoing` | Synthetic recognition that wrongdoing occurred |
| `specific_responsibility` | Synthetic act-focused responsibility appraisal |
| `global_self_condemnation` | Synthetic global negative self-appraisal |
| `exposure_before_others` | Synthetic visibility or imagined exposure before observers |
| `perceived_violation` | Synthetic perception of violation, corruption, or contamination |
| `boundary_threat` | Synthetic perception of boundary threat |
| `perceived_suffering` | Synthetic recognition of suffering, vulnerability, or need |
| `mind_perception` | Synthetic recognition of another as minded and experiencable |
| `admired_moral_action` | Synthetic perception of morally admirable action |
| `perceived_moral_excellence` | Synthetic perception of morally excellent motive or conduct |
| `agency_capacity` | Synthetic capacity to act, repair, or regulate |
| `repair_pathway` | Synthetic availability of confession, restitution, and changed conduct |
| `social_support` | Synthetic supportive relational or institutional condition |
| `guilt` | Synthetic act-focused self-condemning moral emotion |
| `shame` | Synthetic self-focused exposure or global self-condemnation emotion |
| `disgust` | Synthetic boundary-making or violation-response moral emotion |
| `compassion` | Synthetic other-suffering moral emotion |
| `elevation` | Synthetic other-praising moral emotion tied to witnessed excellence |
| `helping_tendency` | Synthetic tendency toward helping or prosocial response |
| `repair_tendency` | Synthetic tendency toward apology, restitution, or correction |
| `avoidance_tendency` | Synthetic tendency toward withdrawal, avoidance, or rejection |
| `constructive_emotion_index` | Synthetic average of guilt, compassion, and elevation |
| `punitive_emotion_index` | Synthetic average of shame and disgust |
