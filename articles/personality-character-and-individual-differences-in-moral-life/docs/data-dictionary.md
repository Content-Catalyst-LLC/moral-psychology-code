# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `case_id` | Synthetic case identifier |
| `broad_personality` | Synthetic broad personality structure or dispositional profile |
| `character_strength` | Synthetic morally valenced character strength |
| `moral_identity` | Synthetic centrality of morality to self-concept |
| `empathy` | Synthetic empathic sensitivity to suffering or vulnerability |
| `conscientious_self_regulation` | Synthetic self-command, reliability, and follow-through |
| `humility` | Synthetic openness to correction and non-defensive learning |
| `self_interest_bias` | Synthetic self-interest bias in moral judgment |
| `situational_pressure` | Synthetic situational or institutional pressure |
| `voice_safety` | Synthetic protection for speaking, warning, dissenting, or reporting |
| `accountability_strength` | Synthetic answerability, review, and responsibility structure |
| `moral_response_profile` | Synthetic moral-response profile |
| `action_latent` | Latent synthetic moral action score |
| `moral_action_probability` | Synthetic probability of moral action |
| `moral_action` | Binary synthetic moral action indicator |
| `character_group` | Character-strength grouping used in summary output |
| `bias_group` | Self-interest-bias grouping used in summary output |
