# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `case_id` | Synthetic case identifier |
| `internalization` | Synthetic private centrality of moral traits to self-concept |
| `symbolization` | Synthetic public expression of moral identity |
| `narrative_integration` | Synthetic integration of moral episodes into self-story |
| `norm_integration` | Synthetic integration of moral norms into self-concept |
| `moral_judgment` | Synthetic moral judgment strength |
| `self_regulatory_capacity` | Synthetic capacity to sustain commitment under pressure |
| `situational_pressure` | Synthetic contextual pressure against moral action |
| `moral_identity` | Synthetic moral identity strength |
| `moral_action_probability` | Synthetic probability of moral action |
| `moral_action` | Binary synthetic moral action indicator |
| `guilt_recognition` | Synthetic recognition of guilt or wrongdoing |
| `humility` | Synthetic openness to correction and limits |
| `accountability` | Synthetic answerability to truth, others, and consequence |
| `defensiveness` | Synthetic self-protective resistance to evidence or correction |
| `repair_capacity` | Synthetic capacity for apology, correction, restitution, and learning |
| `identity_group` | Synthetic high/low moral identity grouping |
| `pressure_group` | Synthetic high/low pressure grouping |
