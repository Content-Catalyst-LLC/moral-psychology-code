# Validation Plan

This scaffold is synthetic and conceptual. Validation therefore focuses on reproducibility, transparency, and interpretive discipline rather than empirical accuracy.

## Checks

1. Scripts run from the article root folder.
2. Output folders are created automatically.
3. Generated tables are written to `outputs/tables`.
4. Figure outputs are written to `outputs/figures` where applicable.
5. Model assumptions are documented.
6. Synthetic variables are not interpreted as measurements of real people.
7. Code comments clearly identify examples as educational scaffolds.

## Possible empirical extensions

Future work could replace synthetic indicators with carefully documented empirical sources, such as:

- cross-cultural moral judgment studies
- organizational ethics surveys
- moral injury scales
- social media moral outrage datasets
- developmental moral learning studies
- public trust and polarization indicators

Any empirical extension should include source documentation, ethical review where appropriate, privacy safeguards, and careful interpretation.
