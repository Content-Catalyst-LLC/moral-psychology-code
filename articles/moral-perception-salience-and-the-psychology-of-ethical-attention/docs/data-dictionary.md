# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `case_id` | Synthetic case identifier |
| `perceived_harm` | Synthetic recognition of injury, suffering, burden, or damage |
| `visible_vulnerability` | Synthetic visibility of dependence, exposure, need, fear, or unequal risk |
| `emotional_activation` | Synthetic affective force that makes the situation morally vivid |
| `mind_perception` | Synthetic recognition that another has feeling, perspective, agency, and experience |
| `contextual_framing` | Synthetic interpretive script that shapes whether the situation appears moral |
| `attentional_competition` | Synthetic distraction, overload, threat, metrics, time pressure, or rival cues |
| `organized_inattention` | Synthetic institutional design factor that makes harm difficult to see or report |
| `latent_salience` | Synthetic underlying moral-salience score |
| `focal_attention_probability` | Synthetic probability that a moral feature becomes focal |
| `focal_attention` | Synthetic binary indicator of focal moral attention |
| `intention_attribution` | Synthetic attribution of intention |
| `reflective_interpretation` | Synthetic interpretive reflection |
| `moral_judgment` | Synthetic downstream moral judgment |
| `repair_pathway` | Synthetic availability of reporting, correction, restitution, or response |
| `ethical_response_tendency` | Synthetic tendency toward ethical response |
