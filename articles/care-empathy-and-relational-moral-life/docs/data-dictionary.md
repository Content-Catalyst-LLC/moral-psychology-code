# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `case_id` | Synthetic case identifier |
| `perspective_taking` | Synthetic perspective-taking capacity |
| `affective_resonance` | Synthetic affective resonance with another person’s state |
| `situational_salience` | Synthetic visibility or salience of another person’s condition |
| `felt_responsibility` | Synthetic felt responsibility to respond |
| `perceived_dependence` | Synthetic perceived vulnerability or dependence |
| `temporal_persistence` | Synthetic capacity to sustain response over time |
| `burnout` | Synthetic burnout or overload pressure |
| `institutional_support` | Synthetic institutional support for care |
| `voice_safety` | Synthetic protection for naming care failures |
| `empathy` | Synthetic empathic activation |
| `caring_response_probability` | Synthetic probability of caring response |
| `caring_response` | Binary synthetic caring response indicator |
| `responsibility_group` | Synthetic high/low responsibility grouping |
| `burnout_group` | Synthetic high/low burnout grouping |
