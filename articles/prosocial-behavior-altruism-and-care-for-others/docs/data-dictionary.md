# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `case_id` | Synthetic case identifier |
| `empathic_concern` | Synthetic empathic concern for another person's welfare |
| `norm_strength` | Synthetic social or moral norm strength |
| `relational_closeness` | Synthetic closeness, relationship, or felt responsibility |
| `visible_vulnerability` | Synthetic visibility of need or vulnerability |
| `perceived_cost` | Synthetic cost of helping |
| `institutional_support` | Synthetic institutional support for prosocial response |
| `prosocial_probability` | Synthetic probability of prosocial action |
| `prosocial_action` | Binary synthetic prosocial action indicator |
| `other_oriented_desire` | Synthetic desire to benefit another for that person's sake |
| `self_oriented_payoff` | Synthetic self-oriented payoff salience |
| `altruistic_motive` | Synthetic altruistic motive strength |
| `attentional_responsiveness` | Synthetic attentiveness to need over time |
| `ongoing_need` | Synthetic ongoing need or dependence |
| `burnout` | Synthetic burnout or depletion pressure |
| `voice_safety` | Synthetic protection for naming unmet care or harm |
| `caregiving_persistence` | Synthetic sustained care response |
