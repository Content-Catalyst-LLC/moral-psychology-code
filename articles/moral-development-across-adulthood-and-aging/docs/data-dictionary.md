# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `case_id` | Synthetic case identifier |
| `age` | Synthetic adult age from 18 to 85 |
| `adulthood_band` | Synthetic adult lifespan band |
| `moral_identity` | Synthetic moral identity integration |
| `prosocial_orientation` | Synthetic care, generosity, and prosocial concern |
| `role_complexity` | Synthetic adult role responsibility across work, family, care, and civic life |
| `moral_emotion_profile` | Synthetic moral-emotional profile |
| `decision_capacity` | Synthetic cognitive-social decision capacity |
| `generativity` | Synthetic long-horizon concern and legacy orientation |
| `caregiving_burden` | Synthetic burden, depletion, or care fatigue |
| `loss_reflection` | Synthetic loss-driven reflection or relational reorientation |
| `voice_safety` | Synthetic institutional protection for warning, dissent, and correction |
| `accountability_strength` | Synthetic accountability, review, and responsibility structure |
| `latent_moral_development` | Synthetic adult moral-development profile |
| `moral_action_latent` | Latent synthetic moral action score |
| `moral_action_probability` | Synthetic probability of moral action |
| `moral_action` | Binary synthetic moral action indicator |
