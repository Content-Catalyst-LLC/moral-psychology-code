# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `case_id` | Synthetic case identifier |
| `perceived_harm` | Synthetic recognition of injury, suffering, degradation, burden, or loss |
| `norm_violation` | Synthetic perceived violation of a duty, rule, standard, or expectation |
| `inferred_intention` | Synthetic judgment about motive, purpose, foresight, or negligence |
| `emotional_activation` | Synthetic affective intensity shaping salience and condemnation |
| `ideological_framing` | Synthetic cultural, ideological, or political interpretation |
| `group_framing` | Synthetic group-based interpretive frame |
| `social_pressure` | Synthetic pressure from audience, group norm, platform, or public context |
| `situational_ambiguity` | Synthetic uncertainty, role conflict, or contextual complexity |
| `perceived_agency` | Synthetic perceived control, knowledge, capacity, and available alternatives |
| `repair_opportunity` | Synthetic availability of apology, restitution, excuse, correction, or repair |
| `dignity_preservation` | Synthetic dignity-preserving response condition |
| `punitive_pressure` | Synthetic pressure toward punishment, humiliation, or sanction |
| `institutional_support` | Synthetic support for reporting, repair, protection, or accountability |
| `latent_wrongness` | Synthetic latent wrongness severity |
| `wrongness_rating` | Synthetic wrongness rating from 1 to 7 |
| `judged_wrong` | Synthetic binary classification of wrongness |
| `latent_blame` | Synthetic latent blame severity |
| `blame_rating` | Synthetic blame rating from 1 to 7 |
| `constructive_response_score` | Synthetic constructive response score |
| `constructive_response` | Synthetic binary constructive response indicator |
