# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `participant_id` | Synthetic participant identifier |
| `culture` | Placeholder synthetic cultural-context label |
| `harm_sensitivity` | Synthetic harm-perception or harm-sensitivity dimension |
| `fairness_sensitivity` | Synthetic fairness, reciprocity, or distribution dimension |
| `loyalty_sensitivity` | Synthetic loyalty or group-obligation dimension |
| `authority_sensitivity` | Synthetic authority, tradition, or norm-order dimension |
| `purity_sensitivity` | Synthetic purity, sanctity, contamination, or symbolic-order dimension |
| `relational_obligation` | Synthetic relational-duty or close-other obligation dimension |
| `norm_learning` | Synthetic norm-learning and socialization dimension |
| `sanction_expectation` | Synthetic expected sanction or enforcement strength |
| `moral_judgment` | Synthetic culturally weighted moral judgment score |
| `norm_enforcement_tendency` | Synthetic norm-enforcement tendency |

## Generated outputs

| Output | Description |
|---|---|
| `cross_cultural_moral_profiles_python.csv` | Python-generated synthetic dataset |
| `cross_cultural_moral_summary.csv` | Python summary by synthetic cultural context |
| `cross_cultural_moral_scenarios.csv` | Python scenario grid across culture, harm, and authority |
| `cross_cultural_moral_high_enforcement_cases.csv` | Synthetic high norm-enforcement cases |
| `cross_cultural_moral_profiles_simulated_data.csv` | R-generated synthetic dataset |
| `cross_cultural_moral_model.csv` | R moral-judgment model coefficients |
| `cross_cultural_moral_model_fit.csv` | R moral-judgment model fit statistics |
| `cross_cultural_norm_enforcement_model.csv` | R norm-enforcement model coefficients |
| `cross_cultural_moral_profile_summary.csv` | R summary by cultural context |
| `cross_cultural_moral_predictions.csv` | R prediction grid |
