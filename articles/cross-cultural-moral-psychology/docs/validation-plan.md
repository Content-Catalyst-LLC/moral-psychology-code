# Validation Plan

This scaffold is synthetic and conceptual. Validation focuses on reproducibility, interpretive clarity, and responsible use.

## Checks

1. Scripts run from the article root folder.
2. Output folders are created automatically.
3. Generated tables are written to `outputs/tables`.
4. Figure outputs are written to `outputs/figures` where applicable.
5. Synthetic cultural contexts are not interpreted as real cultures, nations, religions, ethnic groups, or populations.
6. Moral judgment values are not interpreted as real-person or real-culture scores.
7. Norm-enforcement tendency is not interpreted as a prediction of real enforcement behavior.
8. Code comments and docs clearly identify the data as synthetic.
9. Measurement caution is documented wherever cultural comparison is modeled.
10. Outputs are not used for cultural ranking, profiling, targeting, surveillance, classification, or decision-making.

## Possible empirical extensions

Future empirical work could include:

- validated cross-cultural moral judgment surveys
- measurement-invariance testing
- translated and locally validated instruments
- ethnographic field notes
- developmental norm-learning data
- cross-cultural fairness and cooperation experiments
- moral reasoning about close others
- norm-enforcement and meta-norm datasets
- religious and political moral-world studies
- diaspora and migration moral-learning studies
- mixed-methods cultural comparison

Any empirical extension should include source documentation, local collaboration, privacy safeguards, ethical review where appropriate, measurement validation, and careful interpretation.
