# Validation Plan

This scaffold is synthetic and conceptual. Validation focuses on reproducibility, interpretive clarity, and responsible use.

## Checks

1. Scripts run from the article root folder.
2. Output folders are created automatically.
3. Generated tables are written to `outputs/tables`.
4. Figure outputs are written to `outputs/figures` where applicable.
5. Synthetic weights are clearly described as illustrative, not empirical.
6. Synthetic periods are not treated as fixed historical boundaries.
7. Dominant modes are not treated as claims that one explanatory approach fully defines a period.
8. Diversification is not treated as a validated bibliometric metric.
9. Outputs are not used for author ranking, disciplinary ranking, citation analysis, or claims about real publication volume.
10. Historical interpretation remains primary; computation supports explanation rather than replacing it.

## Possible empirical extensions

Future empirical work could include:

- bibliometric analysis of moral psychology publications
- citation-network analysis
- topic modeling across journals and books
- archival coding of philosophical and psychological traditions
- historical timeline visualization
- cross-series intellectual history mapping
- author and concept network analysis
- institutional and disciplinary genealogy mapping

Any empirical extension should include source documentation, coding rules, uncertainty, bias assessment, and interpretive caveats.
