# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `case_id` | Synthetic case identifier |
| `age` | Synthetic age from 5 to 18 |
| `perspective_taking` | Synthetic cognitive perspective-taking capacity |
| `empathic_concern` | Synthetic empathic concern |
| `norm_understanding` | Synthetic understanding of norms, rules, and obligations |
| `peer_context` | Synthetic peer context and peer learning |
| `family_support` | Synthetic family support and moral socialization |
| `school_climate` | Synthetic school climate and institutional support |
| `peer_status_pressure` | Synthetic peer-status pressure or threat of exclusion |
| `latent_moral_development` | Synthetic moral-development profile |
| `harm_understanding` | Synthetic harm-based moral understanding |
| `fairness_reasoning` | Synthetic fairness reasoning |
| `conventional_sensitivity` | Synthetic conventional-rule sensitivity |
| `personal_autonomy_understanding` | Synthetic personal-domain or autonomy understanding |
| `development_band` | Synthetic developmental pattern classification |
| `moral_action_probability` | Synthetic probability of moral action |
| `moral_action` | Binary synthetic moral action indicator |
