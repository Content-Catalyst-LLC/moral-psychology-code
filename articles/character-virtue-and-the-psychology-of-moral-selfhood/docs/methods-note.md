# Methods Note

This article scaffold uses synthetic data to model character, virtue, and moral selfhood. It includes trait disposition, habituated practice, identity integration, self-regulation, accountability, repair capacity, situational pressure, institutional support, character strength, moral selfhood profile, virtuous action probability, and high-character low-action cases.

## Core methodological claim

Character and virtue should be modeled as structured but situated patterns. Character is not an invulnerable essence, and it is not a fiction dissolved by context. Moral selfhood integrates character, virtue, identity, accountability, repair, habit, and self-regulation across time and situations.

## Modeled constructs

- trait disposition
- habituated practice
- identity integration
- self-regulation
- accountability
- repair capacity
- situational pressure
- institutional support
- character strength
- moral selfhood profile
- virtuous action probability
- high-character low-action cases

## Synthetic design

The synthetic dataset creates case-level values for trait disposition, habituated practice, identity integration, self-regulation, accountability, repair capacity, situational pressure, and institutional support. It then derives character strength, moral selfhood profile, and virtuous action probability. The design intentionally allows high-character low-action cases, where situational pressure and weak support suppress virtue expression despite relatively strong character.

## Interpretation

The generated data are synthetic. They should be used to demonstrate modeling logic, reproducibility, and article-level scaffolding. They should not be interpreted as empirical findings about real persons, students, employees, workplaces, schools, organizations, professions, institutions, cultures, communities, or moral worth.
