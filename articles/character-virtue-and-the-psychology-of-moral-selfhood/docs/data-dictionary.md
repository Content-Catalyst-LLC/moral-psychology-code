# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `case_id` | Synthetic case identifier |
| `trait_disposition` | Synthetic trait-like moral disposition |
| `habituated_practice` | Synthetic practiced moral habit strength |
| `identity_integration` | Synthetic integration of moral commitments into selfhood |
| `self_regulation` | Synthetic capacity to sustain commitment under pressure |
| `accountability` | Synthetic answerability to truth, others, consequence, and correction |
| `repair_capacity` | Synthetic ability to apologize, correct, restore, and learn from failure |
| `situational_pressure` | Synthetic contextual pressure against virtuous action |
| `institutional_support` | Synthetic institutional support for ethical action |
| `character_strength` | Synthetic character strength profile |
| `moral_selfhood_profile` | Synthetic moral selfhood profile |
| `virtuous_action_probability` | Synthetic probability of virtuous action |
| `virtuous_action` | Binary synthetic virtuous action indicator |
| `character_group` | Synthetic high/low character grouping |
| `pressure_group` | Synthetic high/low pressure grouping |
