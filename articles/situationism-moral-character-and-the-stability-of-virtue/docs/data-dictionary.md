# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `person_id` | Synthetic person identifier |
| `situation_id` | Synthetic situation identifier |
| `character_strength` | Synthetic character strength or virtue-relevant tendency |
| `self_regulation` | Synthetic self-regulatory capacity |
| `habit_strength` | Synthetic practiced moral habit strength |
| `moral_identity` | Synthetic moral identity centrality |
| `depletion` | Synthetic fatigue, stress, or self-regulatory depletion |
| `situational_pressure` | Synthetic contextual pressure against moral action |
| `voice_safety` | Synthetic protection for warning, dissent, correction, and truth-telling |
| `accountability_visibility` | Synthetic visibility of answerability, review, and consequence |
| `situation_class` | Synthetic situation-class label |
| `moral_action_probability` | Synthetic probability of moral action |
| `moral_action` | Binary synthetic moral action indicator |
| `pressure_band` | Synthetic situational-pressure grouping |

## Generated outputs

| Output | Description |
|---|---|
| `situationism_character_python.csv` | Python-generated synthetic person-situation dataset |
| `situationism_character_summary.csv` | Python pressure-band summary |
| `situationism_character_scenarios.csv` | Python scenario grid |
| `situationism_high_character_low_action_cases.csv` | Python high-character low-action cases |
| `situationism_character_simulated_data.csv` | R-generated synthetic person-situation dataset |
| `situationism_character_model.csv` | R model coefficients |
| `situationism_character_model_fit.csv` | R model fit statistics |
| `situationism_pressure_summary.csv` | R pressure-band summary |
| `situationism_character_predictions.csv` | R prediction grid |
