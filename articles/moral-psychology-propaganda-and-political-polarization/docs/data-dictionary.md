# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `participant_id` | Synthetic participant identifier |
| `group` | Placeholder synthetic group label |
| `value_profile` | Synthetic value-orientation indicator |
| `trust_allocation` | Synthetic trust allocation across sources or institutions |
| `identity_salience` | Synthetic political or group-identity salience |
| `propaganda_exposure` | Synthetic propaganda exposure indicator |
| `threat_perception` | Synthetic perceived threat indicator |
| `outrage_amplification` | Synthetic outrage amplification indicator |
| `democratic_trust` | Synthetic confidence in shared institutions, evidence, or procedure |
| `cross_group_contact` | Synthetic contact or civic recognition across groups |
| `moral_political_judgment` | Synthetic moral-political judgment score |
| `moral_exclusion_risk` | Synthetic moral exclusion risk value |
| `polarization_distance` | Absolute distance between average group judgments |

## Generated outputs

| Output | Description |
|---|---|
| `propaganda_polarization_python.csv` | Python-generated synthetic dataset |
| `propaganda_polarization_summary.csv` | Group summary |
| `propaganda_polarization_scenarios.csv` | Scenario grid across propaganda, threat, and democratic trust |
| `propaganda_polarization_high_risk_cases.csv` | Synthetic high moral-exclusion-risk cases |
| `propaganda_polarization_simulated_data.csv` | R-generated synthetic dataset |
| `propaganda_polarization_model.csv` | R moral-political judgment model coefficients |
| `propaganda_polarization_model_fit.csv` | R model fit statistics |
| `moral_exclusion_model.csv` | R moral-exclusion model coefficients |
| `propaganda_polarization_group_summary.csv` | R group summary |
| `propaganda_polarization_predictions.csv` | R prediction grid |
