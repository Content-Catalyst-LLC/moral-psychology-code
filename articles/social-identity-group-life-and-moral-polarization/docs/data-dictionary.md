# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `participant_id` | Synthetic participant identifier |
| `group` | Placeholder synthetic group label |
| `value_profile` | Synthetic value-orientation indicator |
| `identity_strength` | Synthetic social or group identity strength |
| `norm_pressure` | Synthetic pressure to conform to group norms |
| `threat_perception` | Synthetic perceived threat indicator |
| `institutional_trust` | Synthetic trust in shared institutions or procedures |
| `cross_group_contact` | Synthetic cross-group recognition or contact |
| `dehumanizing_rhetoric` | Synthetic exposure to dehumanizing rhetoric |
| `loyalty_pressure` | Synthetic loyalty pressure indicator |
| `moral_judgment` | Synthetic moral judgment score |
| `moral_exclusion_risk` | Synthetic moral exclusion risk value |
| `polarization_distance` | Absolute distance between average group judgments |
