# Data Dictionary

## Synthetic variables

| Variable | Description |
|---|---|
| `case_id` | Synthetic case identifier |
| `age` | Synthetic developmental age from childhood through young adulthood |
| `perspective_taking` | Synthetic capacity to consider intentions, viewpoints, and reciprocity |
| `norm_exposure` | Synthetic exposure to rules, norms, dialogue, and social learning |
| `domain_differentiation` | Synthetic distinction among moral, conventional, personal, and prudential domains |
| `relational_orientation` | Synthetic sensitivity to care, dependency, relationship, and responsiveness |
| `authority_orientation` | Synthetic tendency to defer to authority, rule, or external command |
| `peer_cooperation` | Synthetic cooperation, mutual respect, and peer-rule negotiation |
| `cultural_context_support` | Synthetic cultural and institutional support for moral dialogue |
| `latent_reasoning` | Synthetic underlying developmental moral reasoning complexity |
| `reasoning_stage` | Synthetic stage-like ordered reasoning category |
| `justice_reasoning` | Synthetic justice, rights, fairness, reciprocity, and principle dimension |
| `care_reasoning` | Synthetic care, relationship, vulnerability, and responsiveness dimension |
| `conventional_sensitivity` | Synthetic sensitivity to convention, authority, coordination, and local rules |
| `autonomy_orientation` | Synthetic ability to evaluate rules through cooperation and mutual respect |
| `developmental_band` | Synthetic age band for summary reporting |
