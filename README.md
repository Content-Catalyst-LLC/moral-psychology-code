# Moral Psychology Code

Open computational scaffolding for the Moral Psychology knowledge series.

This repository supports article-level examples for moral judgment, moral reasoning, moral intuition, moral perception, ethical attention, moral emotion, conscience, guilt, shame, moral motivation, the judgment-action gap, moral self-regulation, moral identity, prosocial behavior, care, empathy, character, virtue, situationism, moral development, personality, moral disengagement, dehumanization, justice, responsibility, blame, punishment, forgiveness, repair, pluralism, cross-cultural moral psychology, institutions, moral injury, social media outrage, propaganda, and moral psychology methods.

The examples are synthetic and educational. They are designed to support transparent modeling, moral-agency simulations, vignette-analysis scaffolds, reproducible notebooks, and article-linked computational demonstrations. They are not clinical, diagnostic, legal, ethical-adjudication, employment-screening, or automated moral-judgment tools.

## Structure

Each article folder contains:

- `python/`
- `r/`
- `julia/`
- `cpp/`
- `fortran/`
- `c/`
- `rust/`
- `go/`
- `sql/`
- `notebooks/`
- `docs/`
- `data/raw/`
- `data/processed/`
- `outputs/`
